# Minecraft-DDOS
Этой простой DDOS клиент бот для майнкрафт,  который нужно использовать ТОЛЬКО для проверки собственного сервера

## Что это?
Нужен для ддос атаки на свой сервер майнрафт. Мне будет не приятно если вы будете использовать для других сервером. Задача максимально простая, боты заходят с на сервер пишут сообщение и выходят. Никаких логинов и прочего. 

### Важно
У вас должен быть установлен python и nodejs
